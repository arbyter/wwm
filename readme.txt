# Bennungsschema

## /static/css
enthält nur CSS-Datein für das Userinteface

## /static/html
enthält nur html-Datein für das Userinteface

## /static/images
enthält nur Grafiken für das Userinteface

## /static/jslib
enthält nur für das Userinteface notwendige Bibliotheken
_nur_ unkomprimierte Bibliotheken hier ablegen!
die Bennung ist dabei [Bibliotheksname].[Pluginname].[Version].js
 Bsp: jquery.ui.1.8.3.js

## /static/js
von uns geschriebener JavaScript-Code für das Userinteface
die Bennung ist dabei [Bereich].js

## /server.js
Einstellung, die für das Projekt notwendig sind

## /server/
enthält sämtliche Skripte, die der Server braucht

## /server/node_modules/
Fremdplugins

## /server/main.js
Eigentlicher Server

## /data/
Ort, um Daten zu speichern
